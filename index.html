## What is it? Markdown styler loads a file from a URL and styles it as a markdown document. ## How do i use it? Say you have a markdown document at `https://my.site/my/url`. Just append the URL you want to style as parameter to this page like this: ``` https://matteobrusa.github.io/md-styler/?url=https://matteobrusa.github.io/md-styler/README.md ``` ## How does it work? Markdown styler is a simple hack standing on the shoulders of giants: [Strapdown.js](http://strapdownjs.com/) and [cors-anywhere](cors-anywhere.herokuapp.com). ## Can I use it with Dropbox links? Sure! ## Demo please https://matteobrusa.github.io/md-styler/?url=https://matteobrusa.github.io/md-styler/README.md
